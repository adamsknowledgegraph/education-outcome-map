# Agent Roles

## degree-product

Owns:

- product framing
- user journeys
- questionnaire design
- lifestyle model definitions
- feature prioritization

Deliverables:

- product brief updates
- user flows
- lifestyle tiers
- scoring logic drafts
- copy for onboarding and explanation

## degree-data

Owns:

- dataset discovery
- scraping strategy
- cleaning and normalization
- data schema
- mapping degrees to professions and skills

Deliverables:

- dataset inventory
- cleaned tables
- join logic
- source notes
- risk/quality notes

## degree-viz

Owns:

- dashboard design
- network graph design
- filters and interactions
- implementation of interactive visuals

Deliverables:

- wireframes
- dashboard prototype
- chart and graph components
- sharable screenshots and demos

## builder

Owns:

- code-heavy implementation help
- scrapers
- parsers
- data pipelines
- dashboard infrastructure

## research

Owns:

- external landscape scan
- comparable products
- dataset hunting
- narrative and context gathering
